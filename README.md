MagangIn adalah aplikasi absensi magang berbasis mobile yang dirancang untuk membantu peserta magang melakukan check-in dan check-out secara digital.

##  Tujuan
Menyediakan desain UI yang sederhana, konsisten, dan mudah digunakan untuk proses absensi harian.

##  Target User
- Mahasiswa / peserta magang

##  User Flow
1. User membuka aplikasi
2. Login
3. Masuk ke halaman Home
4. Melakukan Check-in / Check-out
5. Melihat Riwayat Absensi
6. Logout

##  Wireframe
Wireframe low-fidelity dibuat untuk memetakan struktur layar dan alur navigasi aplikasi.

##  UI Design
Desain high-fidelity dibuat dengan memperhatikan:
- Konsistensi warna
- Hierarki visual
- Kemudahan penggunaan (usability)
